What is RxJS

lodash for events
// full version
RxJS is a library for composing asynchronous and event-based programs by
using observable sequences. It provides one core type, the Observable,
satellite types (Observer, Schedulers, Subjects) and operators inspired
by Array#extras (map, filter, reduce, every, etc) to allow handling asynchronous
events as collections.

What it can do? 

const source = getAsyncStockData();

const subscription = source
  .filter(quote => quote.price > 30)
  .map(quote => quote.price)
  .subscribe(
    price => console.log(`Prices higher than $30: ${price}`),
    err => console.log(`Something went wrong: ${err.message}`);
  );
// callback version, but only happened once
getAsyncStockData(function(err, quote) {
  if (err) {
    return console.log(`Something went wrong: ${err.message}`);
  }
  var price = quote.price;
  if (price > 30) {
    return console.log(`Prices higher than $30: ${price}`);
  }
});
